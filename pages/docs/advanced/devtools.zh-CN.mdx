# DevTools


<Callout>
  SWRDevTools 不是 Vercel 的官方项目.
</Callout>

[SWRDevTools](https://swr-devtools.vercel.app/) 是 SWR 的开发者工具，能够帮助开发者对 SWR 的 cache 和 fetcher 进行调试。

您可以从扩展页面安装 SWR DevTools 并且在没有任何设置的情况下使用它！

- Chrome: https://chrome.google.com/webstore/detail/swr-devtools/liidbicegefhheghhjbomajjaehnjned
- Firefox: https://addons.mozilla.org/en-US/firefox/addon/swr-devtools/

在安装好，SWR DevTool 面板将出现在浏览器开发者工具中。

在[官方网站](https://swr-devtools.vercel.app/)和 [Github 仓库](https://github.com/koba04/swr-devtools)中获取更多信息！
